# Retail Sales Analytics Dashboard | Power BI

## Project Overview

This project is an interactive Power BI dashboard designed to analyze retail sales performance, customer behavior, product performance, and time-based business metrics. The dashboard provides actionable insights through dynamic KPIs, advanced DAX measures, and interactive visualizations to support data-driven decision-making.

## Dashboard Pages

### Executive KPI Page

* Sales, Profit, Orders, and Customer KPIs
* Business performance overview
* Key insights and recommendations

### Product Performance Page

* Product and category-wise sales analysis
* Top and bottom performing products
* Product contribution to revenue and profit

### Customer Analytics Page

* Customer segmentation
* Repeat vs One-Time Customers
* Average Order Value and Discount Analysis
* Top Customers by Sales

### Time Intelligence Page

* Month-to-Date (MTD), Quarter-to-Date (QTD), and Year-to-Date (YTD) analysis
* Previous Period and Same Period Last Year (SPLY) comparisons
* Sales growth trends and performance monitoring

## Tools & Technologies

* Power BI
* DAX
* Power Query
* Excel

## Key Business Insights

* MTD Sales increased by 6.5%
* QTD Sales increased by 5.5%
* YTD Sales are 2.6% below the benchmark
* Repeat customers contribute significantly to overall revenue
* Product performance analysis identifies key revenue-driving products

## Recommendations

* Sustain recent sales momentum
* Focus on high-performing products and customers
* Improve retention strategies for one-time customers
* Address underperforming periods to improve YTD performance
